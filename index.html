About the Money Burn Monitor

The Money Burn Monitor is a lighthearted yet eye-opening tool designed to calculate the real cost of meetings based on participants' hourly rates.

Why I Built This Tool

I’ve been in way too many disorganized, drawn-out sessions where people talked for hours, yet nothing concrete came out of it. Frustrated by meeting after meeting, I decided to write some code instead of participating. And that’s how this script was born. The first version was created on June 16, 2021.
